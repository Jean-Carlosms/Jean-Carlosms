# Jean Carlos Meira da Silva - React Portfolio

Professional portfolio for Jean Carlos Meira da Silva, a Mechatronics Engineer focused on Industrial Automation, Manufacturing Digitalization, Data Solutions, IIoT and Robotics.

The portfolio presents applied engineering projects, professional experience, core expertise, education, selected certifications and current career direction for opportunities as Manufacturing Engineer, Automation Engineer, Data/Automation Engineer, Robotics Engineer and Industrial Digitalization Engineer.

- React
- Vite
- JavaScript
- CSS
- Lucide React

- About Me
- Selected Projects
- Core Expertise
- Professional Experience
- Education
- Selected Certifications & Training
- Current Direction
- Contact

## Run Locally

```bash
npm install
npm run dev
```

The development server is provided by Vite.

## Build

```bash
npm run build
```

The production build is generated in `dist/`.

## GitHub Pages

This project uses relative asset paths through `base: "./"` in `vite.config.js`, which allows the generated static files to work under a GitHub Pages project path.

Recommended publishing flow:

1. Run `npm run build`.
2. Copy the generated files from `dist/` to the repository `docs/` folder if GitHub Pages is configured to publish from `docs/`.
3. Commit and push the updated source and static files.
4. In GitHub, confirm that **Settings > Pages** points to the correct branch and folder.
